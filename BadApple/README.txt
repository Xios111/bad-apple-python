# Bad Apple!! Black & White Animation

This project plays the famous "Bad Apple!!" animation in a completely black-and-white (shadow art) format, synchronized with music using Python.

## Requirements
Make sure you have the following libraries installed before running the project:
- opencv-python
- pygame

You can install them by running this command in your terminal:
pip install opencv-python pygame

## How to Run
1. Ensure that "bad_apple.mp4" and "bad_apple.mp3" are in the same folder.
2. Run the "play.bat" script.
3. Press the 'q' key on your keyboard to exit the animation.
