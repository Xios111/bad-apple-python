# Bad Apple!! Python & OpenCV Animation

This project plays the famous **"Bad Apple!!"** animation in a completely black-and-white (threshold) format, synchronized with music using Python.

## Features
- Real-time video processing and binary thresholding using OpenCV.
- Synchronized audio playback using Pygame.
- Exit the animation instantly by pressing the 'q' key.

## Requirements

Make sure you have Python installed and the necessary libraries configured before running the project:

```bash
pip install opencv-python pygame
